## PARKING PRIVATE 

Parking private es un programa privado que maneja las ganacias de tu parqueo dandote las facilidades de cualquier otro facilitador de tickets y todo bajo tu propio control total.

Tecnologias usadas:
- PHP 8
- LARAVEL 10
- BACKPACK 6
- BOOTSTRAP 4

para arrancar el proyecto es necesario unicamente ejecutar los siguientes comandos en el orden exacto.

```bash
git clone https://github.com/andygomezb7/ParkingPrivateSystem.git
```
```bash
npm install
```
```bash
php artisan migrate
```
```bash
php artisan serve
```
